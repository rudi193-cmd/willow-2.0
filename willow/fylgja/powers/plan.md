# power: plan
b17: FYLP2 · ΔΣ=42

**When:** Ratified scope that needs multiple steps.

1. Outcome in **one sentence** (done = measurable).
2. Ordered steps — each step is one verifiable action (file/command/decision).
3. Dependencies — DB, env, Sean gate, other repo.
4. Explicit **non-goals** (one line).
5. First step only — what runs first after planning.

**Don’t:** Expand scope mid-plan without new ratification. Don’t bury decisions in prose.
